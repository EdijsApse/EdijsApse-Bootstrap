ITLat WEB programmēšanas kursu pirmā Mājaslapa

Mājaslapa tika izstrādāta gan brīvajā laikā, gan lekciju laikā. Mājaslapas galvenais mērķis ir nostiprināt zināšanas WEB izstrādes pamatos. Izstrādes laikā tika iepazīstināts ar versiju kontroles sistēmu - GIT, un izveidots personīgais profils GITHUB mājaslapā, kurā arī tika veidots mājaslapas versijas

Mājaslapā tiek pielietots:

  • HTML
  • CSS
  • JavaScript

WEB Programmēšanas kursu plāns

Kursu laikā tiek plānots iepazīties ar sekojošajiem tematiem:

  • Skices veidošana
  • Izstrādes rīki
  • HTML5 ievads un to elementi
  • HTML formas
  • CSS ievads un tā sintakse
  • CSS pozicionēšana
  • Izvēlnes veidošanas principi
  • CSS „media” vaicājumi
  • Vizītkartes tipa mājas lapas izveide
  • Ievads par „jQuery” un tā sintakse
  • Ietvara „Bootstrap” ievads un tā pielietošana adaptīva (responsīva) dizaina veidošanā
  • Adaptīva mājas lapas izveide
  • OOP PHP
  • Versiju kontroles sistēmas
  • Tīmekļa servera konfigurēšana un domēna reģistrēšanā
  • Virtuālo hostu izveide
  • Datu bāzes normālformas
  • Satura pārvaldības sistēma „Joomla”
  • Ievads par ietvara „Laravel” un tā pielietošana
  • Google piedāvātie rīki mājas lapām
